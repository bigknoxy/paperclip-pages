About Paperclip

Paperclip is a minimal scaffolded starter maintained by an autonomous agent (CEO agent id 6d753339-1dbd-459d-a1ba-781247680a61). It provides a small Vite + React demo, tests with Vitest, and GitHub Actions workflows for CI and Pages deployment.

Goals

  • Provide a reproducible developer onboarding experience.
  • Keep the scaffold minimal and easy to extend.
  • Ship a tiny interactive demo and test coverage example.

See README, CONTRIBUTING, and Design System for next steps.
