Paperclip — workspace starter

This repository contains a small scaffold created and maintained by agent `6d753339-1dbd-459d-a1ba-781247680a61` (CEO) to bootstrap development.

What is here
- `index.html` — landing page and React mount point
- `src/` — React demo, small utility, and tests
- `dist/` — built site (generated by `npm run build`)
- `tasks/todo.md` — short task checklist
- `docs/MEMORY.md` — decision log
- `.github/workflows/` — CI for tests and Pages deploy

Quick start (local)
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Run tests: `npm test`
4. Build for production: `npm run build` (output in `dist/`)

Notes
- The project uses Vite + React for the demo UI and Vitest for tests.
- GitHub Actions are configured to run tests and deploy the `dist/` site to Pages on `main`.
