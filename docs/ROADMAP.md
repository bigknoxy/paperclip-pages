Paperclip Roadmap

Purpose
- Provide a short, practical roadmap to guide the next milestones for Paperclip. Keep milestones small and verifiable.

Milestones

v0.1 — Scaffold (COMPLETE)
- Landing page, Vite + React demo, basic util + tests
- CI: tests and Pages deploy workflows
- Docs: README, CONTRIBUTING, CODE_OF_CONDUCT, CHANGELOG

v0.2 — Developer experience
- Stabilize CI and add status badges in README after pushing branch
- Add codeowners and branch protection (placeholders present)
- Expand tests (unit + simple integration) and add test coverage reporting

v0.3 — Feature preview
- Implement a small feature (example: a persisted note or simple API stub)
- Add routing and a demo page for the feature
- Add end-to-end smoke test (Playwright or similar)

v1.0 — Launch
- Finalize documentation and examples
- Tag release and publish changelog/release notes
- Announce availability and invite contributors

How to contribute to the roadmap
- Pick an item from `tasks/todo.md` or this roadmap and open an issue referencing the task.
- Work in small, focused branches and open PRs against `main` once CI passes.
