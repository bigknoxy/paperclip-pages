💡 Pro Tips

  • Use this tool to embed small icons and graphics directly in CSS without HTTP requests
  • Optimized SVGs work best - remove unnecessary attributes and minimize whitespace
  • URL encoding is required for characters like <, >, #, and & in CSS
  • Single quotes around the data URI prevent issues with double quotes in SVG attributes
  • For repeated backgrounds, use background-size: contain or cover
  • Keep inline SVGs under 2-3KB for best performance; larger images should be external files
