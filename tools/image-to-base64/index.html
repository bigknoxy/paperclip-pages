Common Use Cases

  • CSS Background Images: Embed small icons directly in your stylesheets to reduce HTTP requests
  • HTML Emails: Many email clients block external images - base64 embeds display immediately
  • Data URIs in APIs: Include images directly in JSON responses without separate file hosting
  • Offline Web Apps: Cache images as data URIs for instant loading without network requests
  • SVG Embedding: Convert SVG files to base64 for broader browser compatibility

Output Format Examples

Data URI

data:image/png;base64,iVBORw0KGgo...

CSS Background

background: url('data:image/png;base64,...')

HTML Image

<img src="data:image/png;base64,...">

Markdown

![Alt text](data:image/png;base64,...)

Tips

  • Base64 encoding increases file size by ~33% - use for small images only
  • Images larger than a few KB should remain as separate files
  • Data URIs work in all modern browsers and most email clients
  • Cached base64 images load instantly since they're part of the document
