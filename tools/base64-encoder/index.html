What is Base64?

  • Base64 is an encoding scheme that converts binary data into ASCII text using 64 characters (A-Z, a-z, 0-9, +, /)
  • Commonly used for data URLs, email attachments, JSON Web Tokens (JWT), and embedding binary in text formats
  • Encoded output is about 33% larger than the original data due to the 6-bit to 8-bit conversion
  • URL-safe Base64 replaces + with - and / with _ to avoid URL encoding issues
