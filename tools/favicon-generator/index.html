What is a Favicon?

A favicon (favorite icon) is a small icon associated with a website, typically displayed in browser tabs, bookmarks, and search results. Modern websites need multiple favicon sizes and formats for optimal display across devices and platforms.

Generated Sizes & Uses

16×16 - Standard favicon for browser tabs
32×32 - New tab page and Windows taskbar
48×48 - Windows desktop shortcuts
180×180 - Apple touch icon for iOS devices
192×192 - Android home screen icons
512×512 - Progressive Web App splash screens
