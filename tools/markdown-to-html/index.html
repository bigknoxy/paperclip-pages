Markdown Syntax Reference

  • Headers: Use # for H1, ## for H2, ### for H3, etc.
  • Bold: Wrap text with **double asterisks** or __underscores__
  • Italic: Wrap text with *single asterisk* or _underscore_
  • Links: Use [text](url) format
  • Images: Use ![alt](url) format
  • Code blocks: Wrap with ```language ... ```
  • Tables: Use | column | column | with separator line
  • Task lists: Use - [ ] for unchecked, - [x] for checked

Learn more at GitHub Flavored Markdown Spec
