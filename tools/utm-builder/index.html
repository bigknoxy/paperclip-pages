UTM Parameter Best Practices

Required Parameters

  • utm_source - Where traffic comes from (google, facebook, newsletter)
  • utm_medium - Marketing medium (cpc, email, social, banner)
  • utm_campaign - Campaign name (spring_sale, product_launch)

Optional Parameters

  • utm_term - Keywords for paid search (running+shoes)
  • utm_content - Differentiate similar links (header_link, sidebar)

Naming Conventions

  • Use lowercase letters consistently
  • Replace spaces with underscores or hyphens
  • Be consistent across campaigns
  • Use descriptive but concise names
  • Avoid special characters except + for spaces in terms

Common Mistakes to Avoid

  • Don't use UTM parameters on internal links (creates self-referrals)
  • Don't forget to URL-encode special characters
  • Don't use different naming conventions for the same source
  • Don't include PII (personally identifiable information)
  • Don't make campaign names too long or complex
