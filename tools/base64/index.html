Convert text to Base64 and back. All processing happens in your browser - nothing is sent to any server.

What is Base64?

  • Base64 is an encoding scheme that converts binary data into ASCII text
  • Commonly used for data URLs, email attachments, and API payloads
  • Encoded output is about 33% larger than the original data
  • Safe for transmission through text-only channels
