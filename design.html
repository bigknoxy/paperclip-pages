Visual Identity Guidelines

Key principles for maintaining consistent branding across Paperclip's ecosystem.

🎨 Color Usage

Always use the primary accent color (#06b6d4) sparingly for CTAs and highlights. Backgrounds should use the dark gradient for visual depth.

📐 Spacing

Maintain consistent spacing using the 4px grid system. Use 16px base spacing, scaling up by factors of 8 for larger gaps.

✨ Effects

Use subtle shadows for elevation. Glow effects on the accent color help interactive elements stand out. Avoid heavy drop shadows.

📱 Responsiveness

Design mobile-first with breakpoints at 768px. Navigation should collapse gracefully on smaller screens.

♿ Accessibility

Maintain WCAG AA contrast ratios. Text primary against background primary provides 12.5:1 contrast ratio.

🔄 Consistency

Reuse existing patterns before creating new ones. All tools should feel like part of the same cohesive ecosystem.
