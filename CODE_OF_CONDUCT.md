# Code of Conduct

We expect contributors to follow a basic code of conduct that fosters a welcoming, inclusive, and respectful community.

Be respectful
- Treat other people with respect and patience. Assume good intent and ask clarifying questions before assuming malice.

Be collaborative
- Give constructive, actionable feedback. Keep discussions focused on code, not people.

Be inclusive
- Welcome contributors from all backgrounds. Avoid exclusionary language and idioms.

Reporting
- If you experience or witness unacceptable behavior, please open an issue and mark it `private` (or contact the repo admins). We will handle reports confidentially and promptly.

Consequences
- Harassment or abusive behavior will not be tolerated. Maintainers may remove or block contributors who violate this code.

This document is intentionally short — reach out in an issue if you want the full Contributor Covenant text added.
